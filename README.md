# 💊 Medicine Store - Full Stack Web Application

A full-stack Medicine Store web application built using **React (Vite)** for frontend and **Spring Boot** for backend.  
This project allows users to browse medicines, add products to cart, and manage product data efficiently.

---

## 🚀 Tech Stack

### 🔹 Frontend
- React.js
- Vite
- JavaScript (ES6+)
- Context API
- Axios
- Bootstrap / CSS

### 🔹 Backend
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- REST APIs

### 🔹 Tools
- Git & GitHub
- Postman
- VS Code
- IntelliJ IDEA

---

## ✨ Features

### 👤 User Features
- View all medicines
- Product details page
- Add to Cart functionality

- Responsive UI

### 🛠️ Backend Features

- Logging implementation

## ⚙️ How To Run The Project

### 🔹 Backend Setup

1. Clone repository
2. Configure MySQL database in `application.properties`
3. Run Spring Boot application

Backend will run on:
```
http://localhost:8080
```

---

### 🔹 Frontend Setup

1. Go to frontend folder
2. Install dependencies:

```
npm install
```

3. Start project:

```
npm run dev
```

Frontend will run on:
```
http://localhost:5173
```
